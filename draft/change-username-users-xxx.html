更改 Windows 中 User 文件夹下的用户名

这篇文章完成日期在：2017-03-05 12:55:43。

Windows 起初在安装的时候会让自己设置一个用户名，手残的自己毫不犹豫的打下了自己的名字。“Vulpoet” 这名字也不错啊，结果仔细一看，多打了个 e。强迫症很不舒服，直接改又不能改文件夹下的名字，然后就…开始改

第一步：启用 Administrator，使用 Administrator 登录

  1. 右键 This PC，点击 Manage
  2. 进入 Local Users and Groups > Users
  3. 点击 Administrator，在属性里保证 Account is disabled 没有被选中
  4. 退出
    然后 Sign out，使用 Administrator 登录

第二步：修改用户名，修改注册表值

  1. 在资源管理器里将自己的用户名文件夹重命名成自己想要的名称
  2. 修改完成后，关闭窗口
  3. 运行 Regedit
  4. 定位到 HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Profilelist
  5. 找到自己的用户名的文件夹，然后修改对应的 ProfileImagePath 值
  6. 修改完成，关闭退出

第三步：完成

  1. 退回自己的账户，重新禁用 Administrator 账户
  2. 修改各种软件的缓存路径 （这一步特麻烦，还得慢慢改）
  3. 全部完成
